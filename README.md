1. Project Name:

    = A8 Books Borrow

2. Project Overview:

    A8 Books Borrow is a modern online book borrowing platform that digitizes the traditional library system. Users can explore a collection of books, view details, and borrow books easily through a clean and user-friendly interface.

3. Purpose:

    The main goal of this project is to:
        1. Simplify the book borrowing process.
        2. Provide a seamless digital library experience.
        3. Practice full-stack development using modern technologies.

4. Live URL

    = https://a8-books-borrow.vercel.app/

5. Key Features:
    1. Browse all available books
    2. View detailed information about each book
    3. Borrow books online
    4. Authentication system (Email/Google login)
    5. Fast and responsive UI
    6. Deployed on cloud for global access

6. Technologies Used:
    Frontend: Next.js, Hero ui, Tailwind CSS, DaisyUI, animate_css
    Database: MongoDB
    Authentication: Better_auth
    Deployment: Vercel

7. NPM Packages Used:
    next js, tailwindcss, daisyui, heroui, react_toastify, animate_css etc
